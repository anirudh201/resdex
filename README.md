# ResDex – Express Frontend + Flask Backend (Dockerized)

## Repository
https://github.com/anirudh201/resdex

## Docker Images
- Backend: docker.io/anirudh72/resdex-backend:1.0
- Frontend: docker.io/anirudh72/resdex-frontend:1.0

## How to Run
1. Clone this repo
2. Run: `docker compose up --build -d`
3. Frontend → http://localhost:3000
4. Backend API → http://localhost:5000
   - GET /list
   - POST /submit

## Notes
- Data stored in backend/data.json
- Demo project only
